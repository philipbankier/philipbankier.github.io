Core Beliefs & Positions

Observability as Contract
Context reads should be explicitly traceable (v1.10). Reserved-field collisions must fail loud, never silent. Observation deserves its own architectural seam (EventSink). You can't optimize what you can't see.
Semantic Extraction is Architectural
LLM-backed extraction deserves a declared seam in the architecture (D025), not inline hacks. One provider configuration point across packs. Semantic attention is data, not just a mechanism — it needs its own observation substrate.
Canonical Type Authority
Action classification is a first-class architectural concern. Types must be canonical at runtime boundaries. Audit trails built into authority systems. (CONTRACT v1.9 / ADR 0016)
Production-Grade Reliability
Record/replay is fundamental for deterministic agent testing. Trial execution should be isolated. Developer overrides must be explicit and receipted — never silent.
Synthesis Proposes, Verdicts Promote
AI generates candidates (identity profiles, project candidates, research findings); only owner verdicts promote to facts. Every proposal must cite evidence refs or is dropped at commit. Determinism floors ensure replayable results. The setup draft is deterministic and editable — owner edits supersede without minting prediction wins. (ADR 0043-0046, Jul 14)
Owner-Consented Research
Web research queries derive only from owner-confirmed facts — emails never enter outward queries. Adaptive multi-round campaigns with query lineage, scope gates, and hard budgets authoritative over model recommendations. Campaign disclosure on the owner-approved plan. (ADR 0040, 0044, Jul 14)
Three-Phase Extraction
Semantic extraction follows prepare (reads) / perform (provider) / commit (writes). Sync default stays byte-identical. Deferred execution mirrors the exact contract. Restart replay must not double-append. (ADR 0041, 0045, Jul 13-14)
Evidence-Authoritative Memory
Compiled memory must augment, never displace, Tier 0/1 evidence. Compiled projections cannot boost their own inputs. Raw sources get full budget and render first. Computed candidates require all cited claims selected. Evaluation regressions directly drive architecture.
Governed Learning → Deterministic Procedures
Dynamic behavior should promote to deterministic procedures through fail-closed gates, guards, and earned-region execution. Deopt and demotion on drift. Typed importance and source-trust vectors. Conversation-outcome attention signals.
Provider-Neutral Connector Control
Connectors should be bounded by operational budgets and a provider-neutral control plane. Conversation families are first-class entities. Evidence-first subject knowledge. Owner-anchored interpretation with outcome-only trust.
Cooperative Scheduling
Runtime should support fair single-writer host scheduling via run quanta. Agents must share execution resources cooperatively, not dominate them.
