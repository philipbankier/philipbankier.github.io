Core Beliefs

Agents should be autonomous and self-improving
BabyAGI, SelfMCP, Regimes demonstrate self-improvement patterns. Dogfooding his own infrastructure.
The log IS the agent
Append-only event log is the source of truth. Graph is a deterministic projection. Enables deterministic replay, cheap forking, end-to-end lineage.
Autonomy requires governance
"The winners in agentic categories will not be the most autonomous systems, but the most governable ones."
Data maturity is required for AI
"AI is forcing a data maturity upgrade. Humans once compensated for fragmentation. Agents cannot."
Build in public to learn faster
"Build in public to learn faster, meet founders, and contribute to the ecosystem."
The simplest approach wins
Recurring theme: "Simplest self-building coding agent", "Simplest self-building general autonomous agent".
AI-augmented development enables self-hosted research
ActiveGraph Lab uses Claude as a code worker to self-repair and improve itself. Believes in self-hosted, local-first AI infrastructure.
Scheduled autonomy should be bounded and killable
ActiveGraph Lab's daily heartbeat advances work inside durable-log, budget, pause, and human-gate constraints.
Graph-native memory should be persistent but semantically honest
ActiveGraph's FalkorDB support persists graph projections while documenting relations-as-nodes and whole-projection read boundaries.
